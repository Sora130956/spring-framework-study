# Spring Framework Study

Spring 官方文档精读笔记，兼顾技术深度学习与英语能力提升。

## 目录

- `core/` — Spring Core（IoC 容器、DI），按章节分子目录（如 `01-ioc-container/`）
- `aop/` — Spring AOP
- `mvc/` — Spring MVC
- `boot/` — Spring Boot
- `templates/` — 笔记模板（`note-template.md`）

## 笔记结构

每篇笔记包含：核心理解 → 关键点（原文+注释+代码）→ 句子解析 → 术语表

## 工作流

1. 阅读文档时复制关键段落 + 简短中文备注
2. 将文档 URL 和复制的内容发给 Claude
3. Claude 整理为结构化笔记，放入对应模块目录
4. Claude 按术语筛选标准提取生僻/专业术语填入术语表
